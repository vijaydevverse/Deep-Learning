## 🧠 Deep Learning Overview

Deep Learning is a subset of Machine Learning that uses **multi-layered neural networks** to learn complex patterns from data.

### Applications:

* Image Recognition
* Natural Language Processing
* Speech Recognition
* Recommendation Systems

---

## 🎯 Classification using Deep Learning

* Used to classify data into categories
* Common in image, text, and tabular data
* Typically uses activation functions like **Softmax** or **Sigmoid**

## ⚙️ TensorFlow

TensorFlow is an open-source framework used for building and training deep learning models efficiently.

---

## 🔢 Tensors

Tensors are the fundamental data structures in deep learning.

### Types:

* **Scalar (0D Tensor)** – Single value
* **Vector (1D Tensor)** – Array of values
* **Matrix (2D Tensor)** – 2D array
* **3D Tensor** – Used in sequences/images
* **Higher-Dimensional Tensors (4D, 5D, etc.)** – Used in complex data like videos

---

### Mathematical Operations:

* Addition
* Subtraction
* Multiplication
* Division
* Matrix Multiplication

### Reshaping & Manipulation:

* `tf.reshape()`
* `tf.transpose()`
* `tf.expand_dims()`
* `tf.squeeze()`

### Aggregation Functions:

* `tf.reduce_mean()`
* `tf.reduce_sum()`
* `tf.reduce_max()`
* `tf.reduce_min()`

### Indexing & Slicing:

* Access specific elements and subarrays

### Broadcasting:

* Perform operations on tensors with different shapes
